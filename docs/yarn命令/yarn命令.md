# yarn 命令

## 安装所有依赖
<pre><code>
    yarn install
</code></pre>

## 安装单个安装包
<pre><code>
    yarn add 包名
    yarn add 包名 -dev
</code></pre>

## 卸载单个安装包
<pre><code>
    yarn remove 包名
</code></pre>

## 更新单个安装包
<pre><code>
yarn upgrade [package] =》npm update - -save 
yarn upgrade [package]@[version] 
yarn upgrade [package]@[tag]
</code></pre>
